# InnoLegend 智能车项目配套资料

欢迎来到 `InnoLegend-Smart-Car-Resources`。

本仓库是我们InnoLegend智能车的配套学习资料仓库，希望能把智能车学习过程中的关键资料尽量整理清楚，帮助大家少走弯路，更高效地完成入门、调试和项目开发。

## 仓库内容简介

当前仓库主要包含以下几部分资料：

### 1. 群光开发板技术手册及使用说明
主要包括：

- 开发板硬件规格书
- 原理图
- IO 分配表
- API 使用说明
- 相关芯片数据手册

适合在你需要了解开发板硬件能力、接口定义、底层资源分配或调用API时查阅。

### 2. 智能车拼车步骤及硬件验证手册
主要包括：

- 智能车拼装教程
- 拼装视频教程索引
- 硬件验证手册

建议在收到套件后，优先按照这一部分内容完成整车拼装与基础硬件自检，避免后续调试过程中因为接线、结构或供电问题影响进度。

### 3. 上位机环境配置步骤
主要包括：

- 上位机软件环境配置说明

建议在开始代码烧录、通信调试、视觉调试之前，先完成上位机环境搭建。

### 4. 代码例程
主要包括：

- 视频课程对应例程
- 电赛F题送药小车相关例程
- 电赛H题巡线小车相关例程

建议结合视频课程与资料文档一起学习。先跑通基础例程，再逐步理解模块功能、接口调用方式与整车控制逻辑。

### 5. 视频课程相关资料
主要包括：

- 课程 PPT
- 与课程章节对应的补充讲义或说明资料

适合在观看B站视频教程时同步参考，用于梳理知识点与查漏补缺。
